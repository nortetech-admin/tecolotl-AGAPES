# Tecolotl AGAPES

Tecolotl AGAPES is an artificial intelligence and drone vision project focused on livestock monitoring, aerial analysis, and future autonomous herding assistance.

The project combines:

- DJI drone integration
- Android mobile development
- Computer vision pipelines
- Dataset generation and annotation
- Machine learning for cattle detection and tracking


## Objective

The main objective of AGAPES is to develop an intelligent aerial monitoring system capable of detecting, tracking, and analyzing cattle from drone footage.

Future stages may include:

- autonomous livestock guidance
- movement analysis
- behavioral monitoring
- smart ranch analytics


## Current Development Stage

The current phase focuses on:

- understanding the DJI Android SDK
- organizing the project architecture
- documenting the development process
- building the dataset pipeline
- extracting frames from drone footage
- preparing annotation workflows


## High-Level Architecture

```txt
DJI Drone
    ↓
Android Application
    ↓
Video/Data Collection
    ↓
Dataset Pipeline
    ↓
Computer Vision Models
    ↓
Detection & Tracking
```

## Main Components

### Drone Layer

Handles aerial navigation, camera access, and live video capture.

### Android Layer

Built using the DJI Mobile SDK for drone communication and control.

### Dataset Pipeline

Processes drone footage into annotated training datasets.

### Vision Pipeline

Responsible for cattle detection, tracking, and future behavioral analysis.

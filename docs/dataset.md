# Dataset

The AGAPES dataset contains aerial livestock footage collected for cattle detection, tracking, and future behavioral analysis tasks.


## Dataset Objective

The main objective of the dataset is to support:

- cattle detection
- livestock tracking
- aerial monitoring
- future autonomous herding research

A future objective is to expand the dataset to include:

- deer detection
- wildlife detection
- animal monitoring for the Coahuila ecosystem

## Data Source

The footage is primarily captured using DJI drones during real livestock monitoring operations.

Frames extracted from the videos are later processed and annotated for computer vision training.
